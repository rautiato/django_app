Personal project: Blog and To-do List Web Apps

Author: Ha Tran

Framework: Django v3.0.6

Languages: Python v3.8.2, HTML5, Bootstrap4, CSS3

Database: SQLite3

Common Features:

  • Register account
  • Login
  • Reset password
  • Update profile, upload profile picture
  • Logout

App: Blog

Features:

  • Create, update, delete posts
  • View a list of posts created by a specific author
  • Access API to get the list of posts in Json format

Rules:

  • Only logged in users can create, update, delete posts
  • The logged in users can only update and delete posts created by themselves

App: To-do List

Features:

  • Add, update, delete a to-do item
  • Mark an item done, the closed items will be crossed off
  • Reopen completed items

Rules:

  • Only logged in users can create, update, delete to-do list
